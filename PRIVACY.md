# Privacy

The framework has no telemetry, accounts, cookies, analytics, or network client. Core APIs retain data only in caller-owned objects. The memory repository is process/page-local; the Web Storage repository writes only when the host explicitly calls it. The static server serves local files and does not intentionally log requests.

Applications built with Ludotape may add their own data collection. Their operators are responsible for disclosure, consent, retention, deletion, and legal compliance. Exported documents and replays can contain author-supplied data; inspect them before sharing.
