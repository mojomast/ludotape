# Support

Read the [documentation index](docs/README.md) and search existing reports first. For usage questions and reproducible defects, open a [GitHub issue](https://github.com/mojomast/ludotape/issues). Include Node version, operating system, exact command, complete error, smallest reproduction, and cartridge identity. Remove private data from documents and replays before sharing. Community support has no service-level guarantee. Security reports belong in the private channel described in [SECURITY.md](SECURITY.md).
